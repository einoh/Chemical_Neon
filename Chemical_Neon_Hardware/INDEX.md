## ?? KEY TERMS

- **HMAC** = Hash-based Message Authentication Code
- **SHA256** = Secure Hash Algorithm 256-bit
- **RFC 2104** = Internet standard for HMAC
- **Rweather** = Arduino crypto library maintainer
- **Nano** = Arduino Nano (ATmega328P microcontroller)
- **Signature** = Cryptographic output (64 hex chars)
